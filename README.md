Код состоит из следующих файлов:
- Text_to_corpus.py -- делает из текста книги (файл Deti_captain_Grant.txt) корпус текстов, разбивая исходный текст на абзацы
- Text_preprocessing.py -- делает предобработку для документов коруса: удаляет стоп-слова, приводит к нижнему регистру, вбирает знаки препинания и цифры, делает лемматизацию
- Три файла для трех видов индексирования: bm25_index.py, word_to_vec_index.py и fasttext.py. Файл bm25_index.py включает функции построения индексов и поиска. Файл word_to_vec_index.py включает функции подготовки предложений для модели, обучения модели, построения векторов, подсчета косинусной близости и поиска. Файл fasttext.py включает функции построения векторов, подсчета косинусной близости и поиска.
- Файл search.py -- главный файл для поиска, из которого всё запускается; импортирует функции из файлов для индексирования. Принимает поиск через терминал с помощью  CLI.

Как искать: ввести в терминале python search.py --interactive. Автоматически выбираеся модель bm25, но можно выбрать другую модель для поиска (с помощью model), ввести запрос. Возращает топ подходящих документов (по этой модели) и время поиска. Также есть команда help, которая подсказывает, какие команды вообще есть.
